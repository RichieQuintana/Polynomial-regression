usando:

3. Model 2 Quadratic Regression with Gradient Descent (from scratch)

En este modelo se implementa descenso por gradiente desde cero para el mismo modelo cuadrtico:

[ y = a_0 + a_1 x + a_2 x^2 ]

Pasos:

  1. Construir la matriz de diseo (X = [1, x, x^2]) usando los datos escalados X_scaled.
  2. Definir la funcin de prdida (MSE).
  3. Calcular el gradiente analtico con respecto a los parmetros (\theta = [a_0, a_1, a_2]^T).
  4. Actualizar parmetros iterativamente:
    ( \theta := \theta - \eta \nabla J(\theta) ).
  5. Mostrar la evolucin del error (MSE) a lo largo de las iteraciones.
  6. Volver a la escala original para comparar con los dems modelos.

def design_matrix_quadratic(x_scaled):
    """Construye [1, x, x^2] a partir de x_scaled (n,1)."""
    x = x_scaled.reshape(-1, 1)
    return np.hstack([np.ones_like(x), x, x**2])


def gradient_descent_quadratic(X_design, y_scaled, lr=0.05, n_iters=2000):
    n, m = X_design.shape
    theta = np.random.randn(m, 1)  # parmetros iniciales aleatorios
    cost_history = []

    for i in range(n_iters):
        y_hat = X_design @ theta
        error = y_hat - y_scaled
        gradients = (2/n) * X_design.T @ error
        theta -= lr * gradients

        cost = np.mean(error**2)
        cost_history.append(cost)

    return theta, cost_history


X_design = design_matrix_quadratic(X_scaled)

def evaluate_and_plot_mlp(model, X, X_scaled, x_plot, x_plot_scaled, label):
    # Prediccin sobre datos originales
    y_scaled_pred = model.predict(X_scaled).reshape(-1, 1)
    y_pred = scaler_y.inverse_transform(y_scaled_pred)

    mse = mean_squared_error(y, y_pred)
    r2 = r2_score(y, y_pred)

    # Curva suave
    y_scaled_plot = model.predict(x_plot_scaled).reshape(-1, 1)
    y_plot = scaler_y.inverse_transform(y_scaled_plot)

    plt.scatter(X, y, label="Datos reales")
    plt.plot(x_plot, y_plot, label=label, linewidth=2)
    plt.xlabel("xi")
    plt.ylabel("yi")
    plt.title(f"Model 3: {label}")
    plt.legend()
    plt.show()

    return mse, r2

Discussion (to be customized with your observations)

  • El modelo analtico sirve como referencia base: es rpido, simple e interpreta fcilmente los coeficientes (a_0, a_1, a_2).
  • El modelo de descenso por gradiente normalmente se aproxima al modelo analtico si la tasa de aprendizaje y el nmero de iteraciones son adecuados.
  • El MLP simple (10 neuronas) tiende a un ajuste casi lineal y puede no capturar bien la curvatura cuadrtica.
  • El MLP moderado (50 neuronas) suele equilibrar bien capacidad de ajuste y estabilidad, logrando un buen (R^2) sin oscilaciones extraas.
  • El MLP complejo (1000 neuronas) tiene tanta capacidad que puede sobreajustar el conjunto de datos: MSE muy bajo en los puntos observados, pero posible comportamiento poco razonable fuera del rango de los datos (extrapolacin).
